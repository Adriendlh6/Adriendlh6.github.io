# Boulangerie Pilotage — version vierge PWA

## Ce que contient cette version
- aucune donnée personnelle
- aucun exemple préchargé
- interface responsive pour mobile, tablette et ordinateur
- installation possible sur iPhone via Safari
- publication simple sur GitHub Pages

## Utilisation locale
Ouvrez `index.html` dans un navigateur moderne.

## Publication GitHub Pages
1. Créez un dépôt GitHub.
2. Envoyez tout le contenu de ce dossier à la racine du dépôt.
3. Activez **Settings > Pages** puis publiez depuis la branche principale.
4. Ouvrez l’URL du site depuis Safari sur iPhone.
5. Dans Safari, faites **Partager > Sur l’écran d’accueil**.

## Important
Les données restent stockées localement sur l’appareil. Utilisez **Sauvegarde complète** régulièrement pour conserver une copie JSON dans Fichiers, iCloud Drive ou Drive.
